Changelog

All notable changes to Meridian, newest first.

v2.1.0 April 14, 2026 LATEST
  • Added
    Natural language query — GA release
    Ask questions in plain English. Meridian translates them to SQL, executes against your indexed datasets, and returns results with an explanation and confidence score. Available on all plans.
    Read the announcement →
  • Added
    Databricks connector
    Connect Databricks SQL warehouses using personal access tokens or service principals. Supports Unity Catalog with row-level security passthrough.
  • Improved
    Dataset indexing speed up to 3× faster
    Parallelized Parquet partition writes and reduced index re-build scope to changed partitions only. 50 GB datasets now index in under 4 minutes (was 12 minutes).
  • Fixed
    Snowflake connector intermittent 401 on session expiry
    Token refresh was not retried on the first failure. Sessions now auto-refresh with a 3-attempt backoff before surfacing an error.
  • Fixed
    Workspace admin could not delete connector created by another user
    Permission check incorrectly required object ownership instead of workspace admin role. Now resolved.
  • Changed
    API rate limits increased for Team and Enterprise plans
    Query API: 120 req/min (was 60) on Team; 600 req/min (was 300) on Enterprise. Ingest API unchanged.
    See updated rate limit docs →
v2.0.2 March 28, 2026
  • Fixed
    Pipeline run history missing after workspace migration
    Run metadata was not re-associated with the migrated workspace ID. Retroactively backfilled for all affected workspaces.
  • Fixed
    CSV connector failing on files with BOM characters
    UTF-8 BOM bytes at the start of a file caused schema inference to fail. BOM is now stripped before parsing.
  • Improved
    Query result pagination now returns a stable cursor
    Previous cursor implementation was offset-based and drifted when new rows were added during pagination. Replaced with keyset-based cursor.
v2.0.0 March 1, 2026
Breaking change: The v1 API base path (/api/v1/) is deprecated as of this release and will be removed on June 1, 2026. Migrate to /api/v2/. Migration guide →
  • Added
    Multi-region workspace support
    Workspaces can now be provisioned in US-East, EU-West, and AP-Southeast. Data residency guarantees available on Enterprise plans.
  • Changed
    Authentication migrated to OAuth 2.0 + PKCE
    Legacy API key auth remains supported for machine-to-machine use cases. Browser-based flows now require OAuth 2.0.
  • Removed
    Legacy CSV bulk upload endpoint removed
    POST /v1/datasets/upload removed. Use the Connector Hub (CSV source type) instead.
v1.9.0 January 15, 2026
  • Added
    Webhook notifications for pipeline events
    Configure HTTP POST callbacks on run.completed, run.failed, and dataset.active events.
  • Improved
    Connector Hub timeout handling
    Source connectors now surface actionable error messages on auth failure, network timeout, and schema mismatch instead of generic 500 errors.
