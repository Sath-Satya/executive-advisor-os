Meridian REST API

Enterprise data platform API. All requests return JSON. Authentication via Bearer token.

Base URL https://api.meridian.io/v2
Authentication: Include your API key as Authorization: Bearer <API_KEY> in all requests. Keys are scoped per workspace.

Datasets

GET /datasets List all datasets
ParameterTypeRequiredDescription
workspace_id string required Workspace to list datasets from
limit integer optional Max results (1–100, default 20)
cursor string optional Pagination cursor from previous response
Response 200
"data": [
  {
    "id": "ds_01hx4g",
    "name": "Sales Q4 2025",
    "rows": 48291,
    "status": "active",
    "created_at": "2025-10-01T09:00:00Z"
  }
],
"next_cursor": "cur_abc123",
"has_more": true
Response 401
{
  "error": {
    "code": "unauthorized",
    "message": "Invalid or expired API key",
    "request_id": "req_7f2k9p"
  }
}
POST /datasets Create a dataset
FieldTypeRequiredDescription
name string required Human-readable dataset name (max 128 chars)
source object required Source connector config (type + credentials)
refresh_schedule string optional Cron expression for auto-refresh
tags string[] optional Organizational tags
Request Body
{
  "name": "Revenue Pipeline 2026",
  "source": {
    "type": "snowflake",
    "warehouse": "ANALYTICS_WH",
    "database": "PROD_DB"
  },
  "refresh_schedule": "0 6 * * *"
}
Response 201
{
  "id": "ds_02ky8m",
  "name": "Revenue Pipeline 2026",
  "status": "initializing",
  "rows": null,
  "created_at": "2026-01-15T14:22:00Z"
}
PATCH /datasets/{id} Update dataset metadata
FieldTypeInDescription
id string path Dataset ID
name string body New display name
refresh_schedule string body Updated cron expression
DELETE /datasets/{id} Delete a dataset (irreversible)

Permanently removes the dataset and all associated pipeline runs. Returns 204 No Content on success.
