Remediation Plan

Remediation Timeline

  • Immediate (0–15 days)
    Execute BAAs with AI inference API and cloud logging sub-processors (GAP-001)

    Legal to draft and route BAAs to both vendors. Escalate to Vendor Management if either vendor delays. No technical work required.

    Owner: Legal / Vendor Management
  • Immediate (0–15 days)
    Update Notice of Privacy Practices to disclose AI processing (GAP-002)

    Compliance to draft updated NPP language for legal review. Publish to member-facing portal and update paper notices in care settings within 30 days of approval.

    Owner: Compliance Officer
  • Short-Term (30–60 days)
    Enable encryption at rest on secondary analytics warehouse (GAP-004)

    Engineering to enable Snowflake Tri-Secret Secure. Validate no performance degradation on reporting workloads. Document configuration change in IT change log.

    Owner: IT Security
  • Medium-Term (60–120 days)
    Implement 7-year AI decision log archival pipeline (GAP-003)

    Engineering to build log export to cold storage (Azure Blob Archive tier). Implement lifecycle policy ensuring 7-year retention. Include retrieval test in DR plan.

    Owner: Engineering / IT Ops
  • Long-Term (120–180 days)
    Implement minimum necessary data scoping for AI inference inputs (GAP-005)

    Engineering to redesign the data pipeline to pass only PA-relevant fields to the AI model, excluding non-essential PHI fields. Requires schema review and model revalidation after input scoping change.

    Owner: Engineering
